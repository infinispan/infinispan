${infinispan.brand.name} -- a distributed, transactional, highly scalable data structure
==========================================================================

This is the ${infinispan.brand.name} distribution packaged as ${wildfly.brand.name} modules.

Requirements
------------

* JDK 8 compliant Java virtual machine.
* ${wildfly.brand.name} installation.

Installation
------------

Copy the contents of the "modules" directory included in this distribution to the "modules" directory of your
${wildfly.brand.name} installation.
