= Configuring Clients for ALPN
Configure clients to use Application-Layer Protocol Negotiation (ALPN) and
Transport Layer Security (TLS) for {brandname} server connections.

.Procedure

. Configure {brandname} server endpoint connectors with encryption.
. Enable ALPN support for Hot Rod clients that use JDK 8.
+
JDK 9 and later supports ALPN by default. For JDK 8, you should use the
`boringSSL` libraries that Netty provide to support ALPN.
