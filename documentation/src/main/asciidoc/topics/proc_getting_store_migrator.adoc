= Getting the Store Migrator
`StoreMigrator` is available as part of the {brandname} tools library,
`infinispan-tools-{infinispanversion}.jar`. You can either build
`infinispan-tools-{infinispanversion}.jar` from source or pull it from the
Maven repository.
