= Model Context Protocol Endpoint

[WARNING]
====
THE MCP ENDPOINT IS AN *EXPERIMENTAL FEATURE* AND MUST BE EXPLICITLY ENABLED.

To enable the MCP endpoint, start {brandname} Server with the `org.infinispan.feature.mcp` system property set to `true`:

.Linux
[source,bash,options="nowrap",subs=attributes+]
----
bin/server.sh -Dorg.infinispan.feature.mcp=true
----

.Windows
[source,bash,options="nowrap",subs=attributes+]
----
bin\server.bat -Dorg.infinispan.feature.mcp=true
----
====

{brandname} server's REST connector exposes an endpoint which implements a Model Context Protocol (MCP) server compliant with the link:https://modelcontextprotocol.io/specification/2025-03-26[2025-03-26 Specification] using the link:https://modelcontextprotocol.io/specification/2025-03-26/basic/transports#streamable-http[Streamable HTTP transport].
