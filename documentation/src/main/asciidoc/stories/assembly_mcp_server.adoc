= Model Context Protocol Endpoint

{brandname} server's REST connector exposes an endpoint which implements a Model Context Protocol server compliant with
the link:https://modelcontextprotocol.io/specification/2025-03-26[2025-03-26 Specification] using
the link:https://modelcontextprotocol.io/specification/2025-03-26/basic/transports#streamable-http[Streamable HTTP transport].
The MCP endpoint is enabled by default on REST connector.
