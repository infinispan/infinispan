= Guide to {brandname} Server Container Image

The {brandname} Server container image runs in a dedicated Java Virtual Machine (JVM) and provides client access to remote caches through Hot Rod, REST, Memcached or RESP (Redis) endpoints.
{brandname} Server speeds time to deployment by separating caches from application logic and offers built-in capabilities for monitoring and administration.
