.SH SYNOPSIS
.B site [
.I --status | --online | --offline
.B ] [
.I cachename.sitename
.B ]
.SH DESCRIPTION
Inspects or changes the status of sites for X-Site replication 
.SH ARGUMENTS
.IP --status
Shows the current status 
.IP cachename
(optional) the name of the cache on which to invoke the specified site command
.IP sitename
(optional for status, required for all other operations) the name of the site
on which to invoke the specified command
