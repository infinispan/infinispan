CD(1)
======
:doctype: manpage


NAME
----
cd - navigates the resource tree


SYNOPSIS
--------
*cd* 'PATH'


DESCRIPTION
-----------
The *cd* command navigates the server resource tree. `PATH` can be either absolute or
relative to the current resource. The special `..` resource name indicates the parent of
a resource.
