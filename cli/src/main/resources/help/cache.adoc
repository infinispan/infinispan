CACHE(1)
========
:doctype: manpage


NAME
----
cache - selects the default cache for subsequent commands.


SYNOPSIS
--------
*cache* ['CACHE_NAME']


EXAMPLE
-------
`cache mycache` +
Selects `mycache` and is the same as navigating the resource tree using `cd caches/mycache`.
